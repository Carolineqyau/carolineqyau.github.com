Web Design Research

May 26th, 2020

This article discusses using responsive images. There has always been this trade-off between using blurry pixelated image on a large display and loadings a huge image on a mobile phone. The picture element and the secret and sizes attribute are drafted into the HTML 5.1 specification which will address the issues:

  • Device-pixel-ratio-based selection
  • Viewport-based selection
  • Art direction-based selection
  • Image format-based selection

The new attribute srcset let us declare a set of image sources, which browsers will serve according to certain conditions we specify using descriptors. x descriptors incidate the pixel-density of the images, while w descriptors indicate the width of the images. The sizes attribute must be included when using srcset with w descriptors.

The picture element wraps the image and its sources. Source provides the browser different versions to choose from when displaying the image according to the device used.

Currently, picture is supported by Firefox, Chrome and Opera whereas Safari and Internet Explorer do not support it natively.
